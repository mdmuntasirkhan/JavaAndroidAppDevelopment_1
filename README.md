# Mobile Programming in java Lab 1

## Overview

This Android app has two screens. The first screen shows your name and a submit button. Clicking the button shows a toast with your name and goes to the second screen, which displays your name and an image. The app supports English and French.

## Features

*   Displays your full name on the first screen.
*   Shows a toast with your name when you click the button.
*   Goes to a second screen with your name and an image.
*   Supports English and French.
*   Uses `strings.xml` for all text.

## Setup

### Prerequisites

*   Android Studio
*   Android emulator or device
*   Basic Java knowledge

### Installation

1.  Clone the repository.
2.  Open the project in Android Studio.
3.  Wait for Gradle to build.
4.  Run on an emulator or device.

## Usage

1.  The app starts on the first screen.
2.  You'll see your name.
3.  Click "Submit".
4.  A toast will show your name.
5.  The second screen will show your name and an image.
6.  Change your device language to French to test French.
